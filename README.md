# meeter-api -- BackEnd
This repository is for the backend of the Meeter page.

Herramientas:
 - TypeScrypt
    TypeScript es JavaSciypt, pero incorpora mejor administración de tipos de datos.
 - Prisma (ORM)
    Sirve para interactuar con la base de datos, hecho para TypeScript.
 - Base de datos (MySQL y MongoDB).
 - Sharp
    Procesamiento de imágenes
    libvips
 - JsonWebToken
    Creación de tokens para establecer sesiones o verificaciones
 - Bcrypt
    Encriptar contraseñas, etc.
 - Dotenv
    Cargar variables de entorno.
 - Nest JS
    Manejo de controladores más organizado.
 - Express
    Para crear el servidor web y recibir peticiones.
 - AI (gemini 2.0 flash)  
    API gratuita con Gemini para el chatbot.
 - Cloudinary
    API con el servidor de Cloudinary para subir imágenes.
 - Cookie-Parser
    Sirve para crear cookies.
 - Cors
    El tema de la restricción de puertos.
 - Nodemailer
    Para enviar e-mails.
 - UUID
    Para crear identificadores UUID.
